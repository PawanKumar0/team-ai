# Research: Technology Stack (Enhanced)

## Core Stack (Confirmed)
- **Frontend**: Next.js 16+, Tailwind CSS 4, Shadcn UI.
- **Backend**: Express (Node.js) API Gateway.
- **AI Core**: LangGraph (Orchestration), LiteLLM (Routing).
- **Persistence**: PostgreSQL (Relational + Checkpointing), Redis (Caching + Rate Limiting).
- **Vector DB**: Pinecone or Qdrant.

## New Research-Driven Additions
- **Sandboxing**: **E2B (Firecracker)** for secure, ephemeral code execution.
- **Observability**: **Langfuse** or **Helicone** for token tracking and trace analysis.
- **Security**: **AgentShield** pattern for adversarial scanning.
- **Quality**: **Plankton** pattern for automated write-time enforcement.
- **Persistence Layer**: **Postgres/Redis Checkpointers** for LangGraph "time-travel" debugging.
