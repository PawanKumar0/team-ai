# Project: Team AI

## Core Objectives
1. **Production Level AI Product**: Build a scalable, enterprise-ready AI Tech Team SaaS.
2. **AI Engineering Transformation**: Transform the user into an AI engineer through JIT (Just-In-Time) learning, explicitly connecting theory (Math, Transformers, System Design) to each sprint.

## Architecture & Stack
- **Frontend**: Next.js (React), Tailwind CSS 4, Shadcn UI.
- **Backend**: Express (Node.js) API Gateway.
- **AI Core**: LangGraph/Agents, LiteLLM (dynamic routing), Super Agent orchestrator.
- **Data**: PostgreSQL (Relational), Pinecone/Qdrant (Vector DB), Redis (Caching/Messaging).
- **Security**: Docker-in-Docker sandboxing for secure code execution.
- **Observability**: AWS/EKS, Helicone/Langfuse (Token tracking), Datadog/Prometheus.

## Roadmap Overview (8 Sprints)
- Sprint 1: Architecture & Repo Setup (In Progress)
- Sprint 2: Core LLM Engine & Dynamic Routing
- Sprint 3: Identity, Auth, & Billing
- Sprint 4: RAG & Long-Term Memory
- Sprint 5: Multi-Agent "Agent as an Employee" Architecture
- Sprint 6: Dynamic UI & Real-Time Dashboard
- Sprint 7: Secure Sandboxing & Execution
- Sprint 8: Production Deployment & Observability
