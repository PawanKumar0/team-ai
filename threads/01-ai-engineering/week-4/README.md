# AI Engineering Week 4

## Focus

- session state
- short-term memory
- long-term knowledge
- summarization
- context control

## Learn

- learn the difference between chat state and retrievable knowledge
- learn when summaries help and when they lose signal
- learn how memory write rules prevent vague magic
