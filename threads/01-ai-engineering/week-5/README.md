# AI Engineering Week 5

## Focus

- orchestration patterns
- MCP
- agent loop
- selective multi-agent

## Learn

- learn the difference between workflow, agent, and multi-agent
- learn when MCP is a better abstraction than ad hoc tool wiring
- learn why A2A is late-stage complexity
