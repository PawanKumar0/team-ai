# Weekly Hub

This file is the 8-week roadmap, not the day-to-day working document.

If you want one place that tells you:

- what to build
- what to learn
- what to design
- what to write
- what to read

use this file.

If you want to actually start the current week, then use:

- [START-EACH-WEEK.md](/Users/pawan/Vizmo/team-ai/threads/00-main-thread/START-EACH-WEEK.md)
- your current `threads/00-main-thread/week-N/README.md`

## How To Use

For the current week:

1. read the week section below
2. open the matching resource pack in [WEEKLY-RESOURCE-PACKS.md](/Users/pawan/Vizmo/team-ai/threads/04-resource-map/WEEKLY-RESOURCE-PACKS.md)
3. open the current week file in `threads/00-main-thread/week-N/README.md`
4. treat that week file as the single working doc
5. open support thread weekly maps only when you need more depth
6. lock the week and start building
